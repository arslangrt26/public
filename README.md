# Simple Interest Calculator

## 📌 Overview
The Simple Interest Calculator is a basic tool that calculates simple interest based on the principal amount, rate of interest, and time period. It helps users quickly determine the interest earned or payable over a given duration.

## 🧮 Formula
Simple Interest is calculated using the formula:

**SI = (P × R × T) / 100**

Where:
- **P** = Principal amount  
- **R** = Rate of interest (per annum)  
- **T** = Time (in years)

## 🚀 Features
- Takes user input for principal, rate, and time
- Calculates simple interest instantly
- Displays accurate results
- Easy to understand and use
- Beginner-friendly project
